# Пати-игры — прогресс

Хаб компанейских мини-игр (заходят с телефонов в одну комнату по коду).
Стек как у «Шпиона»: чистая статика (HTML/CSS/vanilla JS) + Firebase RTDB, GitHub Pages, пуш через прокси 7897.
Ассеты версионируются `?v=N` (**текущая v=15**) — при правках CSS/JS бампать в index.html.

## Ростер (решено)
1. **100 к 1** — ✅ готово, играбельно, тестируется вживую.
2. **Словесный движок** (тумблер: Совпади / Не совпади / На одной волне) — дальше.
3. Одна лёгкая — дальше.

## Статус: 100к1 играбелен, идут итерации по фидбеку

### Что готово
- **Каркас комнат**: вход по 4-символьному коду, автоочистка, автовосстановление сессии по localStorage-id.
- **Presence «серый»**: обрыв → игрок не дропается, гаснет серым (`online:false` через `onDisconnect`), возврат → снова онлайн, слот/очки на месте. Heartbeat `lastSeen` раз в 25с. В таблице лидеров у офлайн-игрока бейдж «не в сети».
- **Хост**: кикает; при обрыве ведущего на 15 мин права уходят самому «старому» онлайн-игроку (транзакция).
- **100 к 1** (пошаговый вскрыв):
  - Хост стартует раунд → вопрос из базы (фильтр `soft`/`hot` тумблером «Острые вопросы 18+»), случайный без повтора.
  - Игроки по кругу пишут догадку → `round/pending` → **хост разбирает** (матчинг), вскрывает ячейку, очки, передаёт ход. Пас — через pending.
  - **Anti-peek**: полное табло только у хоста (localStorage), в БД лишь вскрытые ячейки.
  - Имя отгадавшего — индиго-бейдж в ячейке.
  - **Подсказки**: общий намёк «Подсказка: …» сверху + кнопка хоста «Открыть букву» → открывает случайные буквы
    закрытых ответов (`/round/hintmask/idx`; неоткрытые `_` серые, открытые белые, пунктуация/цифры видны сразу).
    Слово с полностью открытыми буквами автозасчитывается БЕЗ очков (пометка «подсказка», очки muted). Угадать
    частично открытое = полные очки. Управление — только хост.
  - Ход офлайн-игрока ждём; у хоста «Пропустить ход». Конец: всё вскрыто / все спасовали / хост завершил → полное табло + счёт.
  - **Азарт** = таблица лидеров по очкам.
- **UI-допы**: счётчик «Ханго хуй?» (левый низ, синхронный по комнате `/party/rooms/CODE/hango`, транзакция);
  реакции — «Реакция» (правый низ, 24 эмодзи 👍👎+топовые/твич), всплывают у табло, синхрон через `/reactions`, эфемерные.

### Матчинг «по смыслу» (проверен: 136/136)
Нормализация (регистр, ё→е) → токены (3+ без стоп-слов) → сравнение слов (точное/по основе/Левенштейн ≤1) →
пересечение токенов догадки и ответа (текст + ключи), порог 3. Ловит перефраз/склонения/синонимы/опечатки/односложный ввод,
отсекает «вообще другое». Предел: отрицание («никогда не начну» не видит «не»). Правило ключей: не класть слово из вопроса.
